# Decrypto Market Code Assesstment
## Getting Started
This application was designed to be built & run using:
* Java 17
* Maven 3.9.6
* Spring Boot 3.2.3 (including Spring Data JPA)

API routing:
* Apache Camel 4.4.1

Documentation
* Swagger 1.6.4
